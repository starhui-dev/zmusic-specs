# ZMusic Specs

Shared specifications for ZMusic projects.

This repository is the source of truth for cross-repository contracts used by
the ZMusic server plugin, client mod, and future related implementations.

## Scope

This repository is for:

- Plugin message protocols
- Cross-project API contracts
- Compatibility policies
- JSON schemas and protocol test vectors
- Architecture decisions that affect more than one repository

This repository is not for:

- Implementation TODOs
- Single-repository refactoring notes
- User-facing documentation pages
- Release checklists for one implementation

## Status

The repository is being prepared for ZMusic V4 protocol work.

The first planned specification is the ZMusic V4 plugin message protocol.
